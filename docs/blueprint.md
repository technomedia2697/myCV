# **App Name**: Apex Technoscapes

## Core Features:

- Animated Hero Section: Showcases Angelo's title, tagline, and key calls to action ('View Projects', 'Contact Me') with smooth Framer Motion animations.
- Professional About Section: Presents a concise professional biography, highlighting roles as Software Engineer, Founder of Techno Media, and Founder of Zone A, emphasizing specialization in Web Development and Flutter.
- Dynamic Tech Stack Display: Visually represents Angelo's diverse skill set, including Web (HTML, CSS, JavaScript, React), Mobile (Flutter, Dart), and Backend knowledge (Node.js, .NET basics) using modern icons.
- Project Showcase: Dynamically loads and displays project cards from 'src/data/projects.json', each featuring an image, title, description, tech stack, and action buttons for 'Live Demo', 'GitHub', and 'Project Files'.
- Admin Project Management Interface: A simplified '/admin' page allowing Angelo to add, edit, or delete portfolio projects using a form interface, with data persistently stored via Local Storage.
- Comprehensive Contact Section: Provides multiple contact options including email, WhatsApp, GitHub, and LinkedIn, along with a user interface for a contact form.

## Style Guidelines:

- Background color: A deep, subtly bluish dark grey (#171A1C) provides a sleek, modern canvas for the dark mode aesthetic.
- Primary accent color: A vibrant, clear blue (#4293FF) used for calls to action, headings, and key highlights, symbolizing technology and professionalism.
- Secondary accent color: A soft, desaturated cyan-blue (#A0DEEB) used for complementary elements, subtle differentiators, and graphical details, maintaining a premium feel.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif), lending a techy, contemporary, and bold presence.
- Body text and detailed descriptions will use 'Inter' (sans-serif), ensuring readability, a modern feel, and a neutral, objective tone for professional content.
- Code snippets and technical references will be styled with 'Source Code Pro' (monospace) for clarity and authenticity.
- Clean, minimalist vector icons with a modern, professional aesthetic. Icons will primarily be outline-based or subtly filled to maintain a premium and uncluttered look.
- Full-width responsive design with distinct sections separated by ample negative space. Glassmorphism elements will be used on cards and UI components to add a sense of depth and modernity while preserving minimalism through frosted, transparent backgrounds and subtle borders.
- Smooth and elegant animations powered by Framer Motion, including subtle hover effects for interactive elements, graceful section transitions, and dynamic content reveals, enhancing user engagement with a polished, premium feel.
